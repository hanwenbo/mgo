## mgo

Mgo is based on the official library [mongo](https://github.com/mongodb/mongo-go-driver).

<br>

### Example of use

```go
    import "github.com/hanwenbo/mgo"

    // connect mongodb
    db, err := mgo.Init("mongodb://root:123456@192.168.3.37:27017/account")

    // close mongodb
    defer Close(db)
```
